# Air Middle Earth Demo

[![Issues](https://img.shields.io/github/issues/adrianbgo/AirMiddleEarthDemo)](https://github.com/adrianbgo/AirMiddleEarthDemo/issues)
[![MIT License](https://img.shields.io/github/license/adrianbgo/AirMiddleEarthDemo)](https://github.com/adrianbgo/AirMiddleEarthDemo/blob/master/LICENSE)

A fun little project based around the idea that Middle Earth developed an airline. This
project is for use as a demo in Java classes to show a wide variety of different concepts,
including:

- Variables and their attributes
- Conditionals and loops
- Arrays and methods
- Collections Framework
- Object Oriented Programming
- Debugging and unit testing

## Features

- Java-based
- Full CRUD operations for users
- Full CRUD operations for admins
- Login system
- Book flights
- Cancel flights
- Random flight scheduler

**There are 2 levels of access:**

- Passenger
- Administrator

## Usage

When you launch the app, it will open in the terminal. Once the terminal is open,
you will need to create a new user in the system via the registration page.

Once a new user is created, go to the login page, and log in with your credentials.

Once logged in, follow the prompts on the screen to try out any of the functions
in the application.
